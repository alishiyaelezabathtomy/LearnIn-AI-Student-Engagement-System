# LearnIn - AI Student Engagement Monitoring System

## Overview

LearnIn is an AI-based system designed to monitor student engagement during online classes using computer vision and machine learning techniques.

## Objective

The project aims to help teachers understand student attention levels by analyzing non-verbal cues such as facial expressions, eye movements, and drowsiness.

## Technologies Used

- Python
- OpenCV
- Deep Learning
- Machine Learning
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Features

- Face detection
- Student attendance monitoring
- Drowsiness detection
- Engagement analysis
- Real-time feedback for teachers

## Applications

- Online learning platforms
- Educational institutions
- Virtual classrooms
